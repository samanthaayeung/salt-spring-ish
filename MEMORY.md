## Key facts
- Files: `index.html` (everything), `background.jpg` (halftone photo backdrop),
  `island.png` (chunky halftone palm — receipt icon + favicon).
- Numbers: grand total $2,481.13, 7 guests, $354.45/person. Accommodation
  $944.00, Victoria ferry $682.50, Salt Spring ferry $172.55, kayak $682.08.
- Numbers live in three places that don't sync automatically: chart data
  (bottom of file), the stat tiles, and the line items.
- Labels are standardized: chart key = breakdown titles = Accommodation /
  Victoria Ferry / Salt Spring Ferry / Activities.
- Motion suite (desktop only, off for touch + reduced-motion): spring tilt
  (MAX_TILT 4°, stiffness .028, damping .85), bend layer (BEND 1.2), shadow
  swing 12px, cursor-following sheen, ornament-glyph cursor trail. All dials
  are named constants in the tilt script.
- Paper is 88% opaque with a frosted backdrop blur; the shadow is cast around
  a transparent shape so nothing dark sits behind the see-through paper.
- The scalloped ends are cut from the single .receipt element with a CSS mask
  (no separate edge strips — they caused seams/overlap bands when tilting).
- Everything printed on the receipt fades to 82% opacity ("thermal ink") via
  one rule: `.receipt > *:not(.receipt-sheen)`.
- Chart tooltip is custom (floats above the bar, receipt-styled); segments
  lighten on hover so the black one reads. The paper eases flat while the
  cursor is over the chart — a tilted surface throws off hover hit-testing —
  and the tooltip only shows once the paper is flat and still (shared flag:
  window.paperState.flat).
- IMPORTANT: the chart must be built immediately at page load — building it
  on a timer breaks its sizing. Its grow-in is synced to the entrance via a
  scoped animation delay (initial reveal only, so hover stays instant).
- Scroll is locked during the entrance slide and the page always starts at
  scroll zero, so the receipt lands exactly at top-of-page and you can never
  scroll past it into empty background.
- Entrance sequence: photo fades in (0.5s), receipt slides up from below and
  glides to a stop (1.7s, starts at 0.4s, long soft deceleration like light
  paper), chart bars grow in during the slide (start ~0.45s, 0.9s duration),
  cursor tilt only wakes after the slide ends. All skipped for
  reduced-motion users.

## Workflow notes
- Images pasted into chat can be recovered from the session log at
  ~/.claude/projects/.../<session>.jsonl (base64 in message content).
- GitHub Pages deploys take ~45–60s; curl with a cache-buster to confirm.
- The in-app preview of local files re-renders on every screenshot, so
  transient JS state (hovers, trails) never shows in captures — verify via
  JS return values instead.
